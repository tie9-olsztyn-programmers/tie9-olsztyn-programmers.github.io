---
title: Repozytorium (repository)
---

## Definicja - repozytorium to
- miejsce, w którym Git przechowuje:
    - wszystkie pliki projektu i
    - historię zmian.
- Może być:
    - lokalne (na komputerze użytkownika) lub
    - zdalne (np. na serwerze takim jak *GitHub*).


## Tworzenie repozytorium
W terminalu, w folderze, w którym jest mój projekt
- `git init`

## Klonowanie istniejącego repo
`git clone https://github.com/uzytkownik/projekt.git`

## Dodawanie plików do repozytorium
- Dodaje całą zawartość folderu: `git add .`
- Dodaje wybrany plik: `git add <nazwaplik>`
