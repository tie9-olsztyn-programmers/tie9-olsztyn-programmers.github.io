# Liczby pseudolosowe
- Aby skorzystać z losowości w Pythonie, musimy najpierw „zaprosić” odpowiedni moduł do naszego programu.
- Robimy to za pomocą instrukcji `import random`.

## Losowe liczby całkowite - `randint()`
Zacznijmy od liczb całkowitych (integers). Wyobraź sobie klasyczną kostkę do gry.

Funkcja, która nas tu interesuje, to `random.randint(a, b)`.
- `a` to początek zakresu.
- `b` to koniec zakresu.
```python
import random

print(random.randint(1,100))
```

## Losowe liczby zmiennoprzecinkowe - `random()`
1. Podstawa: random.random()

Zwraca liczbę `float` z przedziału `[0.0,1.0)`.

Oznacza to, że `0.0` może zostać wylosowane, ale `1.0` już nie.

```python
import random

szansa = random.random()
print(szansa)
# Wynik np.: 0.8123491...
```

2. Konkretny zakres: `random.uniform(a, b)`

Zwraca liczbę `float` z przedziału `a` do `b`.


:::tip Zaokrąglanie wyniku
- Liczby zmiennoprzecinkowe wyglądają brzydko na wyjściu (np. 38.421991283).
- Na maturze często wymagane jest podanie wyniku z określoną dokładnością.

Używamy funkcji `round(liczba, miejsca_po_przecinku)`.

:::

```python
import random

temperatura = random.uniform(36.6, 40.0)
print(temperatura) # np. 37.36504644900828

ladny_wynik = round(temperatura, 2) # Np. 38.42
print(ladny_wynik) # np. 37.37

```

## Obliczanie liczby `π` metodą Monte Carlo

### Ogólnie
Metoda Monte Carlo polega na modelowaniu procesu losowego w celu obliczenia wartości,
która jest trudna (lub niemożliwa) do wyznaczenia analitycznie.
Na maturze często pojawia się w kontekście obliczania pola powierzchni nietypowych figur lub
właśnie przybliżania liczby PI.


### Częściowa metafora
Wyobraź sobie, że masz kwadrat o boku `2R`,
a wewnątrz niego wpisane jest koło o promieniu `R`.
Jeśli zaczniesz losowo "rzucać rzutkami" w ten kwadrat,
to stosunek rzutek, które trafiły w koło, do wszystkich rzutek oddanych w kwadrat,
będzie w przybliżeniu równy stosunkowi pól tych figur.

### Podział na etapy

1.  **Definicja obszaru**: Określamy granice, w których będziemy losować punkty (np. kwadrat).
2.  **Losowanie**: Generujemy dużą liczbę par współrzędnych `(x, y)`.
3.  **Weryfikacja**: Sprawdzamy, czy punkt `(x, y)` spełnia warunek (np. czy leży wewnątrz koła).
4.  **Estymacja**: Na podstawie proporcji trafień obliczamy wynik końcowy.


### Implementacja w Pythonie

:::info Równanie koła
Warunek `x**2 + y**2 <= 1`:
- to matematyczne równanie koła.
- Jeśli suma kwadratów współrzędnych jest mniejsza lub równa kwadratowi promienia,
punkt leży wewnątrz.

:::

:::info `wynik_pi = 4 * (trafienia_w_kolo / liczba_prob)`
Możnik `4` wynika z przekształcenia wzoru:
- `Pole_Koła = PI * r**2`
- `Pole_Kwadratu = (2r)**2` --> `Pole_Kawadratu = 4r**2`



:::

```python
import random


def oblicz_pi(liczba_prob):
    trafienia_w_kolo = 0

    for _ in range(liczba_prob):
        # 1. Losujemy współrzędne z zakresu [-1, 1]
        x = random.uniform(-1, 1)
        y = random.uniform(-1, 1)

        # 2. Sprawdzamy warunek koła: x^2 + y^2 <= r^2
        # Dla r = 1 wzór to: x**2 + y**2 <= 1
        if x ** 2 + y ** 2 <= 1:
            trafienia_w_kolo += 1

    # 3. Wyliczamy pi na podstawie stosunku pól
    # Pole koła / Pole kwadratu = pi*r^2 / (2r)^2 = pi / 4
    # Zatem: pi = 4 * (trafienia / wszystkie_proby)
    wynik_pi = 4 * (trafienia_w_kolo / liczba_prob)
    return wynik_pi


# Przykład użycia
n = 1000000
print(f"Przybliżenie pi dla {n} prób: {oblicz_pi(n)}")
```
