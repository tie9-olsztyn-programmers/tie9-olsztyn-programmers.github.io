# Struktury Nieliniowe w Pythonie
W odróżnieniu od struktur liniowych (takich jak listy, stosy czy kolejki),
gdzie elementy są ułożone jeden po drugim (sekwencyjnie),
w **strukturach nieliniowych** jeden element może być połączony z wieloma innymi.


Dane te nie tworzą prostego łańcucha, lecz organizują się w:
- **hierarchie** (drzewa) lub
- **sieci** (grafy).


## 🌳 Drzewa
Jest to nieliniowa struktura danych organizująca elementy w sposób hierarchiczny.
Przypomina odwrócone drzewo w naturze – korzeń jest na górze, a gałęzie rosną w dół.

#### Cechy charakterystyczne:
* **Korzeń (*Root*):** Węzeł początkowy, nie posiada rodzica.
* **Dzieci (*Children*):** Każdy węzeł może mieć **dowolną liczbę** dzieci.
* **Liść (*Leaf*):** Węzeł, który nie posiada dzieci.
* **Brak cykli:** Z jednego punktu do drugiego prowadzi tylko jedna ścieżka.



#### Zastosowanie:
* Systemy plików (katalogi i pliki).
* Struktura dokumentów HTML/XML (DOM).



### Drzewa binarne (*Binary Trees*)
- To szczególny przypadek drzewa ogólnego.
- W informatyce jest znacznie częściej wykorzystywane ze względu na przewidywalność struktury,
co ułatwia tworzenie algorytmów rekurencyjnych.

#### Cechy charakterystyczne:
- **Ograniczenie**: Każdy węzeł może mieć maksymalnie dwoje dzieci.
- **Rozróżnialność**: Dzieci mają swoje stałe pozycje – Lewe (Left) i Prawe (Right).
- Drzewo binarne może być puste (None).


#### Zastosowanie:
- Drzewa BST (szybkie wyszukiwanie).
- Drzewa wyrażeń arytmetycznych.
- Algorytmy kompresji (np. drzewa Huffmana).### Gdzie się tego używa? (Zastosowania Maturalne)

### Prosta implementacja
Zdefiniujemy drzewo i przypiszmy im konkretne wartość, aby zamodelować:
użycie dla
```
      10
     /  \
    5    20
   /
  3
```


```python
class Node:
    def __init__(self, value):
        self.value = value      # Przechowywana dana (np. liczba)
        self.left = None        # Lewe poddrzewo (początkowo puste)
        self.right = None       # Prawe poddrzewo (początkowo puste)

    def __repr__(self):
        # Ta metoda jest wywoływana, gdy drukujemy CAŁY obiekt
        return f"Node({self.value})"

    def print_in_order(self):
        # 1. Visit Left Subtree (Safe Check)
        if self.left:
            self.left.print_in_order()

        # 2. Visit Node (Self)
        print(self.value, end=' ')

        # 3. Visit Right Subtree (Safe Check)
        if self.right:
            self.right.print_in_order()


# Create the root
root = Node(10)

# Add children to root
root.left = Node(5)
root.right = Node(20)

# Add child to the left node (grandchild of root)
root.left.left = Node(3)

# Weryfikacja struktury
print(f"Root: {root.value}")             # 10
print(f"Left child: {root.left.value}")  # 5
print(f"Left-Left leaf: {root.left.left.value}") # 3
print(f"Right child-leaf: {root.right.value}")

 ############
# wypisanie #
 ############
root.print_in_order()

## magiczna metod a __repr__
print()
print("-"*20)
node_list = [root.left.left, root.left, root, root.right]
print(f"List: {node_list}")
```
## Kopiec (*Heap*)
:::into Czym jest kopiec?
Kopiec to niemal pełne drzewo binarne, które spełnia warunek kopca:
- *min-Heap* (kopiec typu min):
    - Wartość każdego rodzica jest mniejsza lub równa wartościom jego dzieci.
    - Na szczycie (korzeniu) zawsze znajduje się najmniejszy element.


- *Max-Heap* (kopiec typu max):
    - Wartość każdego rodzica jest większa lub równa wartościom jego dzieci.
    - Na szczycie znajduje się największy element.



:::

#### Min-heap
```
          1                <-- Poziom 0 (Korzeń)
       /     \
      3       5            <-- Poziom 1
     / \     / \
    7   8   10  12         <-- Poziom 2
```

Odpowiadający mu kod:
```python
# Indeksy:      0   1   2   3   4   5    6
kopiec_lista = [1,  3,  5,  7,  8,  10,  12]

"""
Przeanalizujmy element o indeksie 1 (wartość 3) (Dla elementu o indeksie i):
    Jego dzieci:
        Lewe:  2 * 1 + 1 = 3    (wartość 7)     Lewe dziecko: 2 * i + 1
        Prawe: 2 * 1 + 2 = 4    (wartość 8)     Prawe dziecko: 2 * i + 2
    Jego rodzic:
        (1 - 1) // 2 = 0        (wartość 1)     Rodzic: (i - 1) // 2
"""
```

#### Max Heap
```
         100               <-- Największy (Korzeń)
       /     \
     80       90
    /  \     /  \
   10  20   30  40
````

Odpowiadająca implementacja w Pythonie:
```python
# Lista dla Max-Heap:
max_heap = [100, 80, 90, 10, 20, 30, 40]

"""
Przeanalizujmy element o indeksie 2 (wartość 90) (Dla elementu o indeksie i):
    Jego dzieci:
        Lewe:  2 * 2 + 1 = 5    (wartość 30)    Lewe dziecko: 2 * i + 1
        Prawe: 2 * 2 + 2 = 6    (wartość 40)    Prawe dziecko: 2 * i + 2
    Jego rodzic:
        (2 - 1) // 2 = 0        (wartość 100)   Rodzic: (i - 1) // 2
"""

```

## Słowniki (*dict*)
:::note Co to jest słownik Pythonie?
- Słownik to nieuporządkowana kolekcja par klucz-wartość.
- Możesz o nim myśleć jak o prawdziwym słowniku językowym:
    - słowo to klucz,
    - definicja to wartość.

:::

### Kluczowe cechy słownika
- Szybkość:
    - Dostęp do wartości na podstawie klucza zajmuje czas stały `O(1)`.
    - To znacznie szybciej niż przeszukiwanie listy!

- Unikalność kluczy:
    - W jednym słowniku nie mogą istnieć dwa identyczne klucze.
    - Jeśli przypiszesz nową wartość do istniejącego klucza, stara zostanie nadpisana.

- Typy danych:
    - Kluczem może być dowolny typ niezmienny (np. `int`, `str`, `tuple`),
    - wartością może być absolutnie wszystko.

### CRUD na słowniku
```python
# 1. Tworzenie słownika
wyniki = {
    "Ania": 85,
    "Bartek": 92,
    "Cezary": 78
}

# 2. Dodawanie i aktualizacja danych
wyniki["Daria"] = 95  # Dodanie nowej pary
wyniki["Ania"] = 90   # Aktualizacja istniejącej wartości

# 3. Bezpieczne pobieranie danych
# Zamiast wyniki["Ewa"] (co rzuci błąd), używamy metody .get()
punktacja = wyniki.get("Ewa", 0) # Jeśli nie ma Ewy, zwróci 0

# 4. Usuwanie elementu
del wyniki["Cezary"]
```

### Zastosowanie - liczymy częstość
```python
tekst = "matura_z_informatyki_2026"
licznik = {}

for znak in tekst:
    # Jeśli znak jest w słowniku, zwiększ o 1,
    # jeśli go nie ma, ustaw 0 i dodaj 1.
    licznik[znak] = licznik.get(znak, 0) + 1

# Wypisanie wyników
for znak, ilosc in licznik.items():
    print(f"Znak '{znak}' wystąpił {ilosc} razy.")

```
